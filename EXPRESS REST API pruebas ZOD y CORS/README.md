### Iniciar proyecto crea (package.json)

```bash
npm init -y
```

### Instala express

```bash
npm i express
```

### Instala express version exacta

```bash
npm i express -E
```

### Editar package.json para poder usar imports: 

```json
"type": "module",
```

### Instala zod

```bash
npm i zod -E
```

### Instala cors

```bash
npm i cors -E
```

### Instala todas las dependencias a la vez

```bash
npm i express zod cors -E
```
